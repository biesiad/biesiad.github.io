• What is it all about?

    A mobile web app for playing Hot n'Cold using GPS. All you need is a phone with a modern browser and somewhere open to play.

    1. Create a new game
    2. Set the secret spot
    3. Share the link with a friend
    4. Watch them navigate to find it

  • Why should I use it?

    For fun!

  • Why did you create it?

    You guessed it... for fun!

  • Could you add tracking, profiles etc?

    Open an issue on GitHub and let's think about some cool extensions.

  • I have privacy concerns. Is it safe?

    Yes. No personal data is collected — only anonymous traffic data via Google Analytics.
